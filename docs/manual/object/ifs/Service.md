## 构造函数
        
### Service
**系统服务管理对象构造函数**

```JavaScript
new Service(String name,
    Function worker,
    Object event = {});
```

调用参数:
* name: String, 服务名称
* worker: Function, 服务运行函数
* event: Object, 服务事件处理

## 静态函数
        
### install
**安装服务到系统**

```JavaScript
static Service.install(String name,
    String cmd,
    String displayName = "",
    String description = "");
```

调用参数:
* name: String, 服务名称
* cmd: String, 服务命令行
* displayName: String, 服务显示名称
* description: String, 服务描述信息

### on

```JavaScript
    Function func);
```

### emit
**主动触发一个事件**

```JavaScript
Boolean Service.emit(String ev,
    ...args);
```

调用参数:
* ev: String, 事件名称
* args: ..., 事件参数，将会传递给事件处理函数

返回结果:
* Boolean, 返回事件触发状态，有响应事件返回 true，否则返回 false
