快速开发

内置模块丰富，安装即用。无需理解复杂的异步概念，使用同步风格编写异步代码，立即开发高性能 Web 服务。

性能卓越

基于 Google V8 引擎，配合高性能 C++ 内核组件。内置多核支持，充分利用服务器资源，提供极致性能。

社区友好

完全开源，兼容 npm 模块管理。拥有活跃的社区支持，丰富的文档和示例，让你的开发之路畅通无阻。

简单易用

使用同步的方式编写异步代码，逻辑清晰，易于维护。

fibjs 采用纤程（Fiber）模型，避免了回调地狱（Callback Hell），让代码像写同步程序一样简单自然，同时保持异步的高并发性能。
