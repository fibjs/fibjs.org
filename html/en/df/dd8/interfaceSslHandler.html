Detailed Description

ssl protocol converter

Used for converting data stream into ssl protocol. SslHandler is a encapsulation of SslSocket, same as follows in logic:

var ss = new ssl.Socket(crt, key);
function(s){
var s1 = ss.accept(s);
hdlr.invoke(s1);
s1.close();
}
