Usage includes:

"rss": 8622080,
"heapTotal": 4083456,
"heapUsed": 1621800,

}

Notes:

  • rss Physical memory being occupied by current process
  • heapTotal Return v8 engine heap memory size
  • heapUsed Return v8 engine heap memory size in use
  • nativeObjects Return number of memory objects
