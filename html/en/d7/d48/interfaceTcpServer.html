Detailed Description

tcp server object for creating a multi-thread tcp server

Use TcpServer to create a multi-thread tcp server.

function func(conn)
{
var data;
while(data = conn.read())
conn.write(data);
conn.close();
}
new net.TcpServer(8080, func).run();
