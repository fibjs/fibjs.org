Detailed Description

Sandbox, to manage a separate running space.

All code is running inside sandbox, global require uses current sandbox to load modules. Following example limits access to assert modules and add a and b two custom modules.

var vm = require('vm');
var sbox = new vm.SandBox({
a: 100,
b: 200,
assert: require('assert')
});
var mod_in_sbox = sbox.require('./path/to/mod');
