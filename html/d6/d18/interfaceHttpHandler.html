详细描述

http 协议转换处理器

用以将数据流转换为 http 协议消息，创建方式：

var hdlr = new mq.HttpHandler(...);

或者：

var hdlr = new http.Handler(...);

readonly Stats HandlerEx.stats
inherited

查询 WebSocket 包协议转换处理器的工作状态

返回的结果为一个 Stats 对象，结构如下：

{
total : 1000, // 总计处理的请求
pendding : 100, // 当前正在处理的请求
request : 10, // 新建的请求
response : 10, // 发送的响应
error : 100 // 发生的错误
}
