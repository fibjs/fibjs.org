◆ open()

static SeekableStream fs.open ( String  fname,
String  flags = "r" 
)
static

打开文件，用于读取，写入，或者同时读写

参数
fname指定文件名
flags指定文件打开方式，缺省为 "r"，只读方式。支持的方式如下：
  • 'r' 只读方式，文件不存在则抛出错误。
  • 'r+' 读写方式，文件不存在则抛出错误。
  • 'w' 只写方式，文件不存在则自动创建，存在则将被清空。
  • 'w+' 读写方式，文件不存在则自动创建。
  • 'a' 只写添加方式，文件不存在则自动创建。
  • 'a+' 读写添加方式，文件不存在则自动创建。
返回
返回打开的文件对象
