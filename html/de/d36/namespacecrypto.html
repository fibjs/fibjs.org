◆ pbkdf1()

static Buffer crypto.pbkdf1 ( Integer  algo,
Buffer  password,
Buffer  salt,
Integer  iterations,
Integer  size 
)
static

依据 pbkdf1 根据明文 password 生成要求的二进制钥匙

参数
algo指定要使用的 hash 算法，详见 hash 模块
password指定使用的密码
salt指定 hmac 使用的 salt
iterations指定迭代次数
size指定钥匙尺寸
返回
返回生成的二进制钥匙
